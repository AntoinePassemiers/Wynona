# Wynona: a Fully-Convolutional Neural Architecture for Protein Contact Prediction

### User installation

Install package from the root of the repository:
```
make
```

or from the src/ folder:
```
cd src
python setup.py install
```

### Dependencies

- NumPy (>= 1.15.4)
- Scipy (>= 1.1.0)
- matplotlib (>= 3.0.2)
- Cython (>= 0.29.2)
- PyTorch (>= 1.0.0)
